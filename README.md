## 스터디 Convention 📣
폴더는 날짜와 문제 번호, 문제이름으로 구성한다.  
ex) 2024.08.26_2504_문제-이름  
즉, 폴더 이름 형식은 <날짜>_<문제번호>_<문제이름> 이 된다. 문제 이름에 띄어쓰기가 있을 경우 - 로 치환한다.  
폴더 안에는 해당 날짜에 푼 문제를 이름과 문제번호로 각자 올린다.  
ex) 박현호_2504.cpp  
커밋 메시지는 [Solve] Problem_문제번호 로 한다.  
ex) [Solved] Problem_2504
